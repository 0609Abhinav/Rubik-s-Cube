## рҹҡҖ Getting Started

### Prerequisites

| Tool | Version |
|------|---------|
| Node.js | вүҘ 16.x |
| npm | вүҘ 8.x |

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/0609Abhinav/Rubik-s-Cube.git

# 2. Navigate to the React app
cd Rubik-s-Cube/cube-solver

# 3. Install dependencies
npm install

# 4. Start development server
npm start
```

> Opens at **http://localhost:3000** рҹҺү

### Production Build

```bash
npm run build
# Output in cube-solver/build/
```

> вҡ пёҸ **First load note:** The Kociemba solver precomputes lookup tables on startup (~3вҖ“5 seconds). The **Solve** button shows `LoadingвҖҰ` during this time вҖ” this is expected and only happens once per session.

## рҹҸ—пёҸ Architecture

```
cube-solver/src/
в”Ӯ
в”ңв”Җв”Җ engine/
в”Ӯ   в””в”Җв”Җ CubeState.js        вҶҗ Cube state (54-char string), solver, scramble
в”Ӯ
в”ңв”Җв”Җ components/
в”Ӯ   в”ңв”Җв”Җ CubeNet.js           вҶҗ 3Г—3 div-grid face renderer (6 faces)
в”Ӯ   в”ңв”Җв”Җ Controls.js          вҶҗ Buttons + speed slider
в”Ӯ   в”ңв”Җв”Җ TouchPad.js          вҶҗ Mobile touch controls (6 faces Г— CW/CCW)
в”Ӯ   в”ңв”Җв”Җ MoveHistory.js       вҶҗ Scrollable move history with active chip
в”Ӯ   в””в”Җв”Җ StatusBar.js         вҶҗ Fixed bottom status indicator
в”Ӯ
в”ңв”Җв”Җ App.js                   вҶҗ Controller: state, events, async playback
в””в”Җв”Җ index.css                вҶҗ All styles: glassmorphism, fluid layout, animations
```

### Key Design Decisions

| Decision | Reason |
|----------|--------|
| 54-char facelet string as state | Native cubejs format вҖ” zero conversion overhead |
| `async/await` playback loop | Clean cancellation via `playingRef`, speed changes mid-play |
| `clamp()` for sticker size | Single value scales from 24px (mobile) to 42px (desktop) |
| Snapshot for step-back | Replay from snapshot is always accurate, no inverse-move bugs |
| `speedRef` alongside `speed` state | Avoids stale closure in async loop |
